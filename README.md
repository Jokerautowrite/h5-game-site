# 龙虾幸存者（动物题材类吸血鬼幸存者）

一个不依赖构建工具的 HTML5 Canvas 小游戏：你操控一只龙虾在海底躲避/反击动物怪潮，自动攻击、拾取经验气泡、升级三选一强化，尽可能存活更久。

## 运行

最简单：直接用浏览器打开 `index.html`。

如果浏览器限制本地文件加载，建议用本地静态服务器：

```bash
python3 -m http.server 5173
```

然后打开 `http://localhost:5173/`。

## 操作

- 开始游戏：点击“开始游戏”按钮（或按 Enter / 空格）
- 移动：WASD 或方向键
- 暂停/继续：空格
- 重开：R
- 升级选择：数字键 1/2/3 或鼠标点击
- 击倒后：Enter / Y 复活继续；Esc / N 结束本局（也可点击按钮）

## 玩法要点

- 主角：🦞 龙虾（自动发射泡泡弹 + 自动钳击；可通过升级解锁“旋流”）
- 敌人：动物主题（海狸/螃蟹/海蛇/小鲨）会追击你
- 掉落：击败敌人会掉经验气泡（绿色泡泡）
- 升级：经验满自动暂停并出现三选一强化
- 波次：怪物“一波一波来”，中间有短暂间隙
- Boss：每 5 波出现 1 个 Boss（击败会掉大量经验）
- 背景：使用离线背景图 `assets/bg.svg`
- 卡通贴图：`assets/sprites/*.svg`（龙虾/怪物/Boss，透明背景）
